# EasyLogin

A sample application for easy login/signup.

### Features
- Login with a name and password.
- Signup with name, password, and country selection.
- Logout

### Topic covered
- Kotlin (Primary Language)
- Firebase RealTime Database
- MVVM
- Jetpack Compose (UI)
